# ConnectWise Dark Theme
A dark theme for ConnectWise web interface.

![ConnectWise Dark Theme](https://i.postimg.cc/zX4MLTN8/Connect-Wise-Dark-Theme.png)

## Install Instructions
1. Download the ConnectWise Dark Theme to your pc, then extract it.
2. Open the Extension Management page by navigating to chrome://extensions.
3. Enable Developer Mode by clicking the toggle switch next to Developer mode.
4. Click the LOAD UNPACKED button and select the extension directory.
5. Extension should be loaded. You may get prompted to disable developer mode. Ignore this.
